### Exploring exoplanets with Planetary Systems Composite Data - [NASA]([url](https://exoplanetarchive.ipac.caltech.edu/docs/data.html)) 

#### This project explores the PSCompPars dataset using Python, pandas, and Google Colab.  

##### 📊 Dataset
We use the **PSCompPars (Confirmed Planetary Systems Composite Parameters)** dataset from NASA’s Exoplanet Archive.  
It contains information about confirmed exoplanets such as:

- Planet names
- Host stars
- Planet mass
- Orbital period
